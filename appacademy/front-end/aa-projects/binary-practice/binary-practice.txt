Binary to base 10

0b1010 = 10
0b0011 = 3

Binary to hexadecimal:

0b1010 = A
0b0011 = 3

Hexadecimal to base 10:

0xa1 = 161
0xff = 255

Hexadecimal to Binary:

0xa1 = 10100001
0xff = 11111111

Decimal to Binary:

8 = 1000
24 = 11000
255 = 11111111

Decimal to Hexadecimal:

8 = 0x8
24 = 0x18
255 = 0xff

Decimal to ASCII:

65 = A
66 = B
97 = a
98 = b

Hexadecimal to ASCII:

0x41 = A
0x42 = B
0x61 = a
0x62 = b

Binary to ASCII:

0b01000001 = A
0b01000010 = B
0b01100001 = a
0b01100010 = b
